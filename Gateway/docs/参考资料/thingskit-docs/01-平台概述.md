# ThingsKit 平台概述

## 产品简介

ThingsKit 是企业级物联网平台，提供从设备接入、数据管理、规则引擎到可视化展示的一站式服务。支持百万级设备并发，助力企业快速构建智能物联网应用。

## 核心指标

| 指标 | 数值 |
|------|------|
| 设备接入能力 | 100万+ |
| 服务可用性 | 99.9% |
| 响应延迟 | 50ms |

## 三位一体协同架构

### 云端平台
- 设备管理与数据存储
- 规则引擎与业务逻辑
- 可视化展示与大屏
- 支持百万级设备并发接入
- 数据安全保障

### 边缘计算
- 边缘网关部署
- 本地数据处理和决策
- 降低云端压力
- 弱网环境业务连续性保障

### 终端设备
- 多种物联网协议接入
- 适配传感器、控制器、智能终端
- 支持摄像头等视频设备

## 产品线

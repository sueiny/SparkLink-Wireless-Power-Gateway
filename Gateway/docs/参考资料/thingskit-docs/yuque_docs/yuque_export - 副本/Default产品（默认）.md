💡注意：

**Default协议 **包含以下协议：[MQTT产品](https://yunteng.yuque.com/avshoi/v2xdocs/sicuwdofg7gfxel5)、[COAP产品](https://yunteng.yuque.com/avshoi/v2xdocs/ciz19l5pws6vya6p)、HTTP ，用户可以通过任何一种协议上报数据或者命令下发

默认产品不可以修改MQTT的Topic，如有特殊需要修改请参考 [MQTT产品](https://yunteng.yuque.com/avshoi/v2xdocs/sicuwdofg7gfxel5)

:::

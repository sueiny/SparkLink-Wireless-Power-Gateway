---
title: 告警配置
---

告警配置是指对告警联系人，告警通知方式进行配置。

# 新增配置
点击新增告警配置，填入相关信息和告警联系人。

:::color4
💡注意：

告警配置中的通知方式实现需在消息管理里面配置好消息配置和消息模板才能生效。

启用告警配置后，场景联动启用并且配置了告警配置，才会产生告警并通知告警联系人，

未启用状态下才可以进行编辑跟删除，点击编辑，修改基础信息，点击确认。

启用了的告警配置被场景联动使用后，需解除掉绑定或者创建新的配置。 

:::

# 启用配置
启用告警配置，场景联动配置了该告警配置才会生效。

:::color4
💡 提示

告警配置在场景联动中启用后不能进行禁用。

:::

# 删除配置
单项或批量删除告警配置数据。

:::color4
💡 提示

用此告警配置绑定的场景联动存在时，告警配置启用时，都不可删除此数据。

:::

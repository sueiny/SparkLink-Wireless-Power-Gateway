---
title: 角色管理
---

      角色用于归纳平台的菜单权限，是菜单权限的拥有者。将创建好的角色赋予不同的子用户，也就将平台的菜单管理权限分配给了子用户。

:::color4
💡 提示

超级管理员新增角色供平台管员使用，租户管理里面的角色供租户管理员使用，租户管理员新增角色供客户账号使用。（角色权限使用不一致）

:::

---
title: 租户列表
---

租户列表页面显示了指定租户的详细信息。在该页面上，可以查看和维护租户的名称、租户配置、租户的角色、管理员账号、状态和创建时间等信息。

# 创建租户
:::color4
💡 提示

一旦删除租户，则租户管理员等数据全部会被删除。**<font style="color:#DF2A3F;">请谨慎删除。</font>**

租户新增需先新增租户配置和租户角色，当然也可以选择默认的租户配置和租户角色。

:::

对平台的应用配置进行编辑，可修改名称，组织，API权限分配，点击确认保存即可修改成功。

:::color4
💡 提示

当删除应用配置时，如果此应用配置之前调用了API，并在调用记录中存在数据，删除时调用记录也一并进行了删除操作。

:::

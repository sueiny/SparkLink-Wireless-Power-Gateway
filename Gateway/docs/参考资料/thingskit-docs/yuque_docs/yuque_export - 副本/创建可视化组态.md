:::color4
💡 提示

启用模板开启后，选择需要的模板，模板选择的产品自动回显，选择设备进行填入即可。

组态未启用模板时，自定义选择产品，组态引用模板后编辑不可取消启用模板。

组件所绑定的设备如果未设置为公开，则数据不显示出来，控制组件操作也没有权限，设置为公开后，数据展示效果才能呈现。

:::

# 导入导出
组态导入导出的功能主要是允许用户将模板从一个可视化组态中移动到另一个ThingsKit平台的可视化组态中。

## 导入功能演示
登录测试服务器，然后创建一个组态；进入设计页面；单击“导入”按钮；选择导入的XML文件。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1757039677317-4dffee00-4892-4226-977e-c2a24fa6dca4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1920" title="" crop="0,0,1,1" id="udb554779" class="ne-image">

平台导入组态案例参考：

[组态案例](https://yunteng.yuque.com/avshoi/v2xdocs/oriscotqwehl9fae)

## 导出功能演示
在服务器中，找到需要导出的组态；进入设计页面；单击“导出”按钮；保存导出的文件，即完成导出操作。

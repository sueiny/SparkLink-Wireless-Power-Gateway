title: 规则链的使用

:::color4
💡注意：

规则链是系统平台核心，每个设备与规则链都有不可分割的关系。**建议：**设备在比较多或者比较杂的情况下，创建多个不同的规则链来进行数据处理，减少默认规则链的处理压力。

:::

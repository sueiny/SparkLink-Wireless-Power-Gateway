# v2.0.0 Release
:::color2
<font style="color:#585A5A;">发布日期：2025/08/25    代码标签：v2.0.0_Release</font>

:::

## 🥇 新增功能
+ 国标级联（支持上级、下级国标级联）
+ 视频回放、录像计划（用户可以通过录像计划生成对应的视频回放并查看）
+ Modbus_TCP协议接入
+ 支持跨设备计算字段（例如：计算A设备和B设备的温度差值）
+ 系统默认的过滤器（配合数据源标准接口使用）

## 🥈 功能优化
+ 前端UI页面进行重构,同时升级vben admin到5.5.4版本
+ 看板进行重构优化
+ 数据中心报表配置功能优化
+ 云边协同同步优化
+ 国标GBT28181优化（点播优化）
+ Modbus RTU优化物模型上报时的解析（只要从任务中心下发的命令，平台根据上报的数据自动解析到对应的物模型）
+ Modbus RTU枚举优化，支持字符串
+ 产品物模型支持拖拽排序
+ 支持批量设备生成
+ 用户、组织优化
+ Node-RED支持https部署
+ 升级ThingsBoard内核版本到4.1.0（jdk17、spring boot 3.2.12）
+ 移除了平台直接将摄像头rtsp转flv的能力，统一使用流媒体进行转码处理
+ 告警弹窗信息优化
+ 告警记录列表详情展示优化

:::info
💡注意：

由于版本升级内容较多，不支持从1x版本升级到2x版本。建议v2.0.0版本部署到新的业务中去使用。

:::

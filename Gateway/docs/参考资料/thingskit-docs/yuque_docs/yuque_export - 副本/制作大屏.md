title: 制作大屏
---

大屏是一种有效的可视化工具，可将收集到的数据以直观、生动的方式展示出来。通过创建交互式的数据大屏，决策者可以快速浏览关键指标，实时了解设备或业务运营状况。这有助于决策者更快地制定决策，并更好地理解数据驱动的洞察。

# 新增大屏
进入可视化管理大屏界面，点击“新增大屏”，在弹出的新增大屏框中，**填写大屏名称**和**选择所属组织及产品，大屏模板可选择设计好的也可不选择进行自主设计**；最后单击 “**确认**”按钮，大屏数据新增成功。

# 大屏设计
接下来，点击<font style="color:rgb(51, 51, 51);">大屏设计，即可进入设计页面：</font>

<font style="color:rgb(51, 51, 51);">在设计大屏界面拖拽图表、信息、小组件等组件进行大屏设计，设计完成并保存内容和保存缩略图，刷新大屏数据界面，缩略图显示为自己设计的图形。</font>

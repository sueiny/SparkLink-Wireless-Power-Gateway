title: 平台通知
---

<font style="color:rgb(51, 51, 51);">     在ThingsKit物联网平台内部向自己管理的用户发送平台通知。</font>

# 新增通知
<font style="color:rgb(51, 51, 51);">点击新增通知，填入想要发送的通知基本信息，确认发布通知。</font>

:::color4
💡<font style="color:rgb(51, 51, 51);"> 提示</font>

<font style="color:rgb(51, 51, 51);">超管账号发送通知为全部，则超管和平台管理员可查看。</font>

<font style="color:rgb(51, 51, 51);">租户账号下(全部就是所有用户账号，包括租户账号)，选择发送到组织的话就是选择的组织下面所关联的账号才能收到通知(发布通知后点击查看可以看到哪些账号有查看权限) 。</font>

<font style="color:rgb(51, 51, 51);">发布后的通知权限只有查看跟删除。</font>

:::

# 发布草稿
通知保存为草稿，只有当前账号可见，可以编辑和删除，编辑时把草稿点击为发布通知即可发送到各个账号，我的通知记录里面可以查看。

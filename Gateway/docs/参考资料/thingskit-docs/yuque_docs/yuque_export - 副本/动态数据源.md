---
title: 动态数据源
---

<font style="color:rgb(5, 7, 59);background-color:rgb(253, 253, 254);">接口动态数据源是指在应用程序的接口层实现数据源的动态选择和切换。这种机制使得应用程序能够根据请求的不同或系统状态的变化，灵活地选择最合适的数据源进行数据处理。</font>

# <font style="color:rgb(5, 7, 59);background-color:rgb(253, 253, 254);">创建数据源</font>
:::color4
💡 注意

如果url里面含有路由参数，示例：http://192.168.1.11:8080/api/test1{testId1}，下方路由参数会自动将testId1作为key填入，value值会在url地址里进行替换，路由参数为必填。

数据来源：平台和第三方，当选择第三方时，没有路由参数选项。

当选择标准接口时，简单的分页接口会直接将参数默认填写好，方便用户使用。

大屏中使用数据源可参考：[附：大屏常用组件使用](https://yunteng.yuque.com/avshoi/v2xdocs/dfd9oz3iwmv4yxte)

:::

进入动态数据源功能，点击创建，填入必填项参数，进行接口测试，获取数据后点击确认新增即可。

## 概述
ThingsKit物联网平台支持通用的MQTT、CoAP、HTTP、TCP/UDP(Modbus rtu/自定义)、GBT28181协议接入您的设备，接入设备类型分为网关设备（如：RTU、DTU、边缘网关等）、网关子设备（接入到网关设备下面的设备，如：温湿度传感器）和直连设备（带有通信能力的设备，如智能一体机），接入前需要确认您的设备类型。

### 物联网网关常见的接口
通过物联网网关接入设备，首先需要了解您采购的网关上面都有哪些接口，常见的接口如下：

> + 数字量：DO、DI
> + 模拟量：AO、AI
> + 串口协议：RS232、RS485
> + 网络口：RJ45
>

### 通过数字量DO和DI能接哪些设备？
#### DO能控制的设备
:::color4
DO作为数字量的输出，简单的理解就是，控制设备的：《开》或《关》

:::

对于大电流，380V的大型设备就非常多；如水泵、风机、三相电机、电焊机、烘干机、热水器、制冷设备、雕刻机、电梯等；这类设备，可以采用继电器、接触器来辅助控制。

对于家用220V小于5A的设备，也就是小于1100W的设备，可以直接控制；如电视机、电风扇、电饭煲、音响、电灯、洗衣机等...

另外就是小电流9~36V设备，俗称弱电类，这类设备也非常多；如阀门开关、电动道闸、声光报警器、指示灯、门禁、甚至包括重启摄像头...

---

## 摄像头接入
**ThingsKit物联网平台支持国标GB/T 28181接入摄像头**

国标GB/T 28181协议是一套我国设计的视频监控系统联网的协议标准，支持国标的摄像头可以使用GB/T 28181协议连接NVR、监控平台系统，并且实现丰富的摄像头管理功能，如摄像头的PTZ控制，双向对讲等功能，在国标GB/T 28181协议中都有完善的定义和体验。

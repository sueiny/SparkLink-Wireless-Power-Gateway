# 如何创建一个自己修改的规则链？
## 答案
:::color3
💡提示

有时候默认规则链功能无法满足当前需求场景下的一些功能，为了实现其需求，可以对规则链进行一些修改，但为了保证原有规则链不会被影响到，可以新增一条规则链在其基础上修改（默认规则链上修改）。

:::

点击导出根规则链（导出前请勿修改）：

<img src="https://cdn.nlark.com/yuque/0/2026/png/36214471/1774595111681-14575fec-df0f-45b7-b5ea-e19653f784d0.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_73%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1329.8700804572882" title="" crop="0,0,1,1" id="u759dc7fd" class="ne-image">

点击导入规则链后选择刚刚导出的文件：

<img src="https://cdn.nlark.com/yuque/0/2026/png/36214471/1774595368317-d5a6ad46-e7e7-4c86-9ccb-8bc1d72dae74.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_71%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1302.8570944479993" title="" crop="0,0,1,1" id="ud1380922" class="ne-image">

效果如下：

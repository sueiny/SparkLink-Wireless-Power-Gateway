# 可视化组件的构成
在构建看板中，选择合适的可视化配置；目前包括几大类：

## 文本组件
文本组件，主要作用是直接展示传感器数据值。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012560484-6e26faff-1e42-4892-8b4e-46d2d51431e3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1916" title="" crop="0,0,1,1" id="u0f7d9f19" class="ne-image">

## 仪表组件
仪表组件，主要是对传感器数据用可视化的仪表来展示，如扇形，圆形等；使得显示不同的效果。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012617393-2c0e3e29-ed5e-4ee3-a95f-9b8d08396554.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1913" title="" crop="0,0,1,1" id="u1c7c45ba" class="ne-image">

## 控制组件
控制组件，主要是针对设备的控制操作提供，如灯光的开关、电机的启停、空调温度设置等。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012655178-7ae4691b-21a2-4315-9df4-ad85afaee409.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1917" title="" crop="0,0,1,1" id="ub0710221" class="ne-image">

## 地图组件
地图组件，主要是针对GPS传感器的数据进行展示，目前提供了实时轨迹和历史轨迹展示。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012765375-c3f1dc5f-766f-4bee-9026-b91d7175cd0f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1908" title="" crop="0,0,1,1" id="u378e87cd" class="ne-image">

## 计量计组件
计量计组件，主要是针对液位传感器和温度传感器的展示组件。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012743631-c64b9c19-58e3-419d-adc4-1b5d4c5f694e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1912" title="" crop="0,0,1,1" id="uaa1e48c7" class="ne-image">

## 统计
统计组件，主要针对设备属性历史数据和实时数据的展示组件。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012797485-f94ad636-3058-428f-b26c-25ddf2f5fb69.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1912" title="" crop="0,0,1,1" id="u6f9f4be1" class="ne-image">

## 告警
告警组件，组件一为显示设备是否有告警，组件二为设备所有告警记录。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1758012844410-0f59aa6a-ef32-4de4-b50f-d820bdee1f78.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1912" title="" crop="0,0,1,1" id="u71033bc3" class="ne-image">

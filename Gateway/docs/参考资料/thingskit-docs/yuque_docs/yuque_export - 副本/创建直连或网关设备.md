💡注意：

网关设备通过**v1/gateway/telemetry**主题上传网关子设备的数据时，如果网关子设备在平台不存在，则会 **自动创建（MQTT）**，自动创建的网关子设备默认使用的 **名称为default **的产品。如需使用自定义的产品需要提前创建好网关和网关子设备，批量添加设备可通过 [批量导入设备并预制访问凭证](https://yunteng.yuque.com/avshoi/v2xdocs/xsbtk69uoogf7w7b) 查看具体操作

:::

# Modbus设备
:::color4
💡注意：

地址码一定要注意填写的是否正确，否则影响数据的正确解析。同时注意地址码切换到的是**HEX**还是**DEC**

如果设备使用的产品，鉴权方式为**注册包 **，那么设备的 **访问凭证 **需要与 **注册包一致**，设备才能正常上线，设备访问凭证不知如何修改，请通过 [获取设备访问凭证](https://yunteng.yuque.com/avshoi/v2xdocs/sixb3ckuygfb7e25#ij0CA) 查看。

:::

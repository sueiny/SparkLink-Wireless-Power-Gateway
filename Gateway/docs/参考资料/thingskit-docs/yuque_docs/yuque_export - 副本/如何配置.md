# 配置方式
      报表配置方式分为立即执行和定时执行，定时执行通过配置相应规则定时执行。

## 定时配置方式
| 参数 | 描述 |
| --- | --- |
| **每日** | 即每天几点定时执行，例如：每天15点 |
| **每周** | 即每周这一天几点定时执行，例如：每周二这一天15点 |
| **每月** | 即每月这一天几点定时执行，例如：每月这一天15点 |


## 数据类型配置方式
| 参数 | 描述 |
| --- | --- |
| **原始数据** | 顾名思义就是没经过计算直接表示的数据，例如温度24 |
| **聚合数据** | 就是通过一定规则进行计算得出的数据，例如：每天15点到16点，按5分钟的间隔时间，进行平均值计算得出的数据 |

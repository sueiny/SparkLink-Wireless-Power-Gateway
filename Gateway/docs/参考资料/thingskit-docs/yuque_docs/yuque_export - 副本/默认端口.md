# 端口开放结果验证
## 第1/3：启动服务端(监听)
:::info
💡 提示

nc命令默认的传输协议是TCP，可以增加参数**<font style="color:#FFFFFF;background-color:#000000;">-u</font>**<font style="background-color:#000000;"> </font>切换为UDP协议。

:::

```shell
nc -l -p 8088
```

```shell
nc -lu -p 8088
```

## 第2/3：启动客户端
```shell
nc   公网IP 公网端口
```

```shell
nc -u 公网IP 公网端口
```

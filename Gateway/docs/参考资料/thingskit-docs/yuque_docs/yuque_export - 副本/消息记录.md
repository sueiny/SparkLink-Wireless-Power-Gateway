title: 消息记录
---

# 概述
平台消息发送记录，例如：告警信息发送记录。

# 前提条件
消息配置正确，消息模板正确，能正常测试使用。

# 消息发送记录
:::color4
💡 提示

其他发送记录页面包括阿里云，腾讯云短信，阿里云语音，钉钉，企业微信。 

用途方式四种：登录，告警通知，忘记密码，清除密码。短信才支持全部，其余只支持告警通知。

:::

# MQTT网关子设备
:::color4
💡注意：

MQTT网关子设备，支持两种创建方式，如下所示：

    - **自动创建**
        * **优势：**不需要人员手动维护，系统根据网关设备上报的数据格式，自动确定网关子设备的名称
        * **缺点：**自动创建的网关子设备，默认使用名称为** default **的产品，如要使用创建的产品需要手动批量修改所属产品
    - **手动创建**
        * **优势：**提前绑定了关系，并使用了自定义创建的产品
        * **缺点：**少量设备需通过手动创建或批量设备导入通过[批量导入设备并预制访问凭证](https://yunteng.yuque.com/avshoi/v2xdocs/xsbtk69uoogf7w7b) 操作

:::

# Modbus网关子设备（Modbus-RTU、Modbus-TCP）
:::color4
💡注意：

地址码一定要注意填写的是否正确，否则影响数据的正确解析。同时注意地址码切换到的是**HEX**还是**DEC**

网关子设备的地址码，应与网关地址码有差异，数据才能正确解析。

例如：网关设备使用地址码（HEX）为**55，**网关子设备地址码（HEX）为** 01**

:::

---
title: 平台定制
---

      每个租户都可以对平台进行自定义配置，包括公司基础信息、平台名称、版权等。

:::color4
💡 提示

租户只有企业信息这一页签，超级管理员含有所有页签（企业信息、平台信息、应用信息）。

:::

# 定制企业信息
企业信息界面填入相关信息，点击更新基本信息即可。

:::color4
💡 提示

超级管理员企业信息与平台管理员相通，租户管理员企业信息与用户账号相通。

:::

---
title: 设备分布
---

      设备分布是指物联网平台中各种设备的地理位置和分布情况。设备分布通过地图形式展示，以便用户能够清晰地了解设备的分布情况。

:::color4
💡 提示

设备新增时未选择地理位置则点击此设备时不能显示设备信息和历史数据。

:::

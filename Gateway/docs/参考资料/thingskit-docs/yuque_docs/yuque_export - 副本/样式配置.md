# 概述
看板样式配置是指通过设置设计器中各个组件的样式属性，来改变它们的表现形式和外观。每个组件都有一些样式属性，如颜色、单位等。通过改变这些属性，可以改变组件的外观。

:::color4
💡 提示

由于看板中，每个组件的样式都不一样，这里就不一一举例，准备了几个组件的样式配置。

:::

## 案例一：文本组件
组件样式设置有：数值字体颜色、图标颜色、数值单位、图标、☑是否显示设备名称。

# GBT28181协议接入（设备、国标平台）
## 平台配置
:::color4
💡提示：

创建好产品和设备后，编辑设备的访问令牌（需要自行设置，可按照示例编辑设备凭证，每一个摄像头访问令牌唯一）。摄像头接入的访问令牌示例：**34020000001320000002**。平台设备访问令牌应该与摄像头<font style="color:rgb(77, 77, 77);">SIP用户名和SIP用户认证ID（或设备编号）保持一致。</font>

:::

## 摄像头操作
摄像头视频推荐配置：

[设备接入问题](https://yunteng.yuque.com/avshoi/v2xdocs/wct0d83bgvmzsrr9#BlPMH)

:::color4
💡注意：

使用GBT281281协议接入，首先需要接入摄像头满足GBT28181协议（2016版本、2022版本）。不同型号摄像头配置方式可能略微有不同。

:::

💡提示：

SIP默认配置：

SIP_PORT（内网端口）：5060，SIP_ID（服务器ID）：51010700599000000001，SIP_DOMAIN（服务器域）：5101070059，SIP PASSWORD（密码）：61332286。

## 萤石云
:::color4
💡注意：

如果提示 **60019 加密已开启**,则需要[关闭设备加密](#M0Uy1)

萤石云常见错误码参考：[https://open.ys7.com/help/78](https://open.ys7.com/help/78)

拥有了萤石云的平台账号，并完成了实名认证。如需注册请参考：[https://open.ys7.com/cn/s/index](https://open.ys7.com/cn/s/index)

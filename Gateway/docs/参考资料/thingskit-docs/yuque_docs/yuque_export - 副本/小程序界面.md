### 登录(仅限小程序第三方登录)
用户通过在PC端创建的账号进行登录，也可以进行微信绑定，快捷登录，可以通过账号绑定的手机号接受验证码进行登录，用户忘记密码可以进行密码重置。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36214471/1764231211533-8fc5f057-d24c-4f61-a7f0-48050d5461a7.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_16%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="329.14285714285717" title="" crop="0,0,1,1" id="uda8fcfd4" class="ne-image">

点击微信头像,绑定账号,输入账号密码即可；

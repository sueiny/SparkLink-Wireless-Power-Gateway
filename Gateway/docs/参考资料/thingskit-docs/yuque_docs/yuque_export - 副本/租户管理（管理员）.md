# 概述
ThingsKit物联网平台是一个多租户管理平台，以租户为平台的运营单元，多个租户之间的数据相互隔离，可通过租户配置和租户角色限制租户的资源使用。

:::warning
💡 提示

本功能只有平台超级管理员才有权限使用。

:::

# 1、概述
本节旨在概述编程界面的结构及其各部分的功能。若您已对这些内容有所掌握，可选择跳过本节。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761552272364-d9983a10-68bd-4909-b82c-221c348f3f94.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1912" title="" crop="0,0,1,1" id="u9c80def9" class="ne-image">

+ **菜单栏**：顶部菜单栏集成了部署按钮和主菜单项，其中流程的导入与导出功能也位于此菜单栏中。
+ **节点库**：汇集了多种预封装功能节点，方便用户直接使用。
+ **工作区**：通过拖放节点并连接它们以表示数据流，用户可以轻松实现自定义编程。
+ **调试区**：包含五个功能页面，分别为**信息窗口**、**帮助文档窗口**、**调试窗口**、**配置节点窗口**和**全局变量窗口**，以便于进行程序调试和配置。

# 2、菜单栏
## 2-1、部署
流程开发完成后，它将仅存在于工作区，并不会自动在系统上执行。要使流程运行，您需要在完成流程设计后，点击‘**部署**’按钮，以便将流程保存并激活在系统中。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761552553833-179be76b-ec80-4fd5-a6ef-80d468089a1d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1900" title="" crop="0,0,1,1" id="ud777ccad" class="ne-image">

## 2-2、菜单
菜单栏提供了**导入**和**导出**等常用功能。流程以JSON文件形式存储，用户可以在流程完成后利用**导出功能**进行备份。同样，若需在新系统中部署现有流程，可使用**导入功能**。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761552808923-c703f9ab-c70b-45a4-8964-744f83cdd781.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1910" title="" crop="0,0,1,1" id="u00bd2370" class="ne-image">

此外，系统默认提供了一些示范流程，存储于本地，以便用户快速开始使用。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761553408554-830ee843-d13a-454d-8e80-de6655f5945c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1912" title="" crop="0,0,1,1" id="u470ac780" class="ne-image">

# 3、节点库
节点库收录了所有已安装且可供直接使用的节点。

节点库根据功能进行分类，依次为：通用节点、功能节点、网络节点、序列化节点、解析节点、存储节点、ThingsKit节点、。后续章节我们将详细介绍每个类别节点的功能和操作方式。

节点通过相互连线构建流程，它们是流程构建的基础和核心。对各个节点功能的熟练掌握直接影响到开发流程的效率。相同的需求可以通过不同的节点组合来实现，不存在唯一的解决方案。我们的目标是持续开发新的节点，以不断提升您的工作效率。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761553662954-0a1543ab-6729-4a26-b78a-f36459fcadac.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1908" title="" crop="0,0,1,1" id="ua576f14f" class="ne-image">

# 4、工作区
## 4-1、流程
概述：流程是由多个节点通过线条相互连接而成的。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761555127423-abf5b81a-efc8-4884-9aa7-516c028967c3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1910" title="" crop="0,0,1,1" id="u738628f7" class="ne-image">

## 4-2、节点
概述：左侧列表中的每个小方块，都代表一个节点；只是每个小方块的功能不一样。

使用时，需要将左侧的小方块，拖入工作区。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761555289303-2a8c6ad2-4dee-45ae-942b-27e2f2e49297.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1911" title="" crop="0,0,1,1" id="u9d9fee5d" class="ne-image">

## 4-3、连线
概述：是指一个节点与另一个节点的关联关系，当线条连接在一起后，程序则可按照您连接的线条进行下一步的执行。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1761555410142-e7296392-9b5c-42f3-b38f-d58230fcf304.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1911" title="" crop="0,0,1,1" id="u2748d965" class="ne-image">

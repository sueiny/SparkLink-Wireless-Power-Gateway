# MQTT协议数据上报
## 直连设备和网关设备
遥测主题：**v1/devices/me/telemetry**

**数据格式A**

数据格式A上报入库的时间以 服务器时间 为准。

```json
{
    "CurrentTemperature":26.8,
    "CurrentHumidity":65.9,
    "APValue":990
}
```

```json
[{
    "CurrentTemperature":26.8,
    "CurrentHumidity":65.9,
    "APValue":990
},
{
    "CurrentTemperature":26.9,
    "CurrentHumidity":66,
    "APValue":995
}]
```

**数据格式B**

数据格式B上报入库的时间以 ts 为准。这种格式推荐在进行 **数据补录** 时使用。

```json
{
	"ts": 1755174230208,
	"values": {
		"CurrentTemperature": 26.8,
		"CurrentHumidity": 65.9,
		"APValue": 990
	}
}
```

## 网关子设备
:::color4
💡注意：

网关子设备数据，其本身不具备网络上报能力，需通过网关设备进行的数据上报，所以MQTTX工具输入的设备凭证是网关设备的凭证。

:::

遥测主题：**v1/gateway/telemetry**

**数据格式A**

数据格式A上报入库的时间以** 服务器时间 **为准**。**

```json
{
  "MQTT测试网关子设备1":[
    {
      "temperature":36.6
    }
  ]
}
```

```json
{
  "MQTT测试网关子设备1":[
    {
      "temperature":36.6
    }
  ],
  "MQTT测试网关子设备2":[
    {
      "temperature":35.6
    }
  ]
}
```

**数据格式B**

数据格式B上报入库的时间以** ts **为准。这种格式推荐在进行** 数据补录 **时使用**。**

```json
{
  "MQTT测试网关子设备1": [
    {
      "ts": 1755517355236,
      "values": {
        "temperature": 42,
        "humidity": 80
      }
    },
    {
      "ts": 1755517387548,
      "values": {
        "temperature": 43,
        "humidity": 82
      }
    }
  ],
  "MQTT测试网关子设备2": [
    {
      "ts": 1755517387548,
      "values": {
        "temperature": 42,
        "humidity": 80
      }
    }
  ]
}
```

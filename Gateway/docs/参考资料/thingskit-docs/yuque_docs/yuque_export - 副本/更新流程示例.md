title: 更新流程示例
---

平台<font style="color:rgb(33, 37, 41);">固件或者软件分配给设备或产品会触发更新过程，更新的进度会保存在设备属性中。</font>  
<font style="color:rgb(33, 37, 41);">平台把固件的通知从队列中推送给设备并转换为以下“共享属性”的更新：</font>

| 共享属性 |
| :--- |
| <font style="color:rgb(33, 37, 41);">fw(sf)_title - 名称</font> |
| <font style="color:rgb(33, 37, 41);">fw(sf)_version - 版本</font> |
| <font style="color:rgb(33, 37, 41);">fw(sf)_size - 文件大小</font> |
| <font style="color:rgb(33, 37, 41);">fw(sf)_checksum - 文件属性</font> |
| <font style="color:rgb(33, 37, 41);">fw(sf)_checksum_algorithm - 校验算法</font> |


# 准备工作：
<font style="color:rgb(33, 37, 41);">下载MQTT客户端示例：</font>

[<font style="color:rgb(33, 37, 41);">链接</font>](about:blank)<font style="color:rgb(33, 37, 41);">：</font>

[mqtt_firmware_client.py](https://yunteng.yuque.com/attachments/yuque/0/2025/py/36222522/1750659188651-1546367e-d4d3-44b7-8cb0-1c575693e6ac.py)

地址，端口，设备token都设置成固定的了，如果需要自行输入，则将红框上面注释取消掉。

<img src="https://cdn.nlark.com/yuque/0/2024/png/36222522/1710989482405-cf5ad819-b893-4b5c-ab94-488e1139f54e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_30%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1047" title="" crop="0,0,1,1" id="u5f79ec5c" class="ne-image">

<font style="color:rgb(33, 37, 41);">在终端中执行以下命令安装MQTT和校验和库：</font>

```json
pip3 install paho-mqtt mmh3 --user
```

<font style="color:rgb(33, 37, 41);">输入客户端文件夹的路径并为平台执行以下命令以获取固件示例脚本：</font>

```json
python3 mqtt_firmware_client.py 
```

执行后打开mqtt_firmware_client.py终端。

<font style="color:rgb(33, 37, 41);">收到以下消息后步骤如下：</font>

Please write your ThingsBoard host or leave it blank to use default (localhost): - 本地主机或平台地址；

Please write your ThingsBoard port or leave it blank to use default (1883): - 按“enter”继续或输入端口号；

Please write accessToken for device: - 复制设备accessToken并将其粘贴到终端；

Please write firmware chunk size in bytes or leave it blank to get all firmware by request: - 为留空则立即以完整大小下载文件如果要分段下载请输入块的大小<img src="https://cdn.nlark.com/yuque/0/2024/png/36222522/1710989831709-aec4ff4a-d0ea-40f9-bb39-8585d16d34ae.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_27%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="948" title="" crop="0,0,1,1" id="u303e824e" class="ne-image">

Firmware is updated！表示此设备固件或者软件升级更新成功。

在保存的<font style="color:rgb(33, 37, 41);">MQTT客户端示例</font>文件夹里面，更新后的固件记录会保存到里面。

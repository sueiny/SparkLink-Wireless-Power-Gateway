title: 意见反馈（管理员）

# 概述
用于接收移动端的用户需求反馈，默认只有超级管理员才有权限操作。

:::color4
💡 提示

本功能只有平台超级管理员才有权限使用。

:::

# 快捷进入
点击详情，查看移动端意见反馈信息。

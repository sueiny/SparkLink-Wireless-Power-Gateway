# 前置条件
需要先[创建产品](https://yunteng.yuque.com/avshoi/v2xdocs/bgaszp6g8gmmvah1)，对物模型属性、服务、事件概念有一定了解。具体可以通过[基本概念](https://yunteng.yuque.com/avshoi/v2xdocs/to3rigum9gyaukab#xCOHe) 查看。

事件分为信息、告警、错误三种类型，详细说明如下所示：

    1. **信息：**是设备上报的一般性通知，如完成某项任务等
    2. **告警、错误：**是设备运行过程中主动上报的突发或异常情况，优先级高。

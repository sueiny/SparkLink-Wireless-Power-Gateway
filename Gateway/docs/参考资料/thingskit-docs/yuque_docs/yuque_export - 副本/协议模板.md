## 云边协议模板同步
边缘环境连接云端时，边缘实例在线。

<img src="https://cdn.nlark.com/yuque/0/2026/png/36222522/1776159039022-5d51634a-a420-4281-833c-8007a7d8fcb8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1909" title="" crop="0,0,1,1" id="ua80e3991" class="ne-image">

<img src="https://cdn.nlark.com/yuque/0/2026/png/36222522/1776159231039-cc23b972-66da-4df0-87c3-d4a41dbcb833.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1910" title="" crop="0,0,1,1" id="ub0a1a091" class="ne-image">

导入后，云端会同步协议模板数据到边缘环境中，这个过程不会太久。

现在协议模板数据已经从云端自动同步成功了，数据会随云端环境变动而变动。

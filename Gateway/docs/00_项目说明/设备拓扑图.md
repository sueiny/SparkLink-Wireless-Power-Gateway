## 数据流

```
RS485设备 → DTU节点(采集) → DTU网关(MQTT上报) → ThingsKit平台
```

## MQTT Topic

| Topic | 用途 |
|-------|------|
| v1/devices/me/attributes | 网关属性 |
| v1/gateway/connect | 子设备上线通知 |
| v1/gateway/telemetry | 子设备遥测（DTU代发） |

# 设备拓扑图

## 物理拓扑（RS485总线）

```
                    ┌──────────────────┐
                    │     DTU网关      │
                    │   (GATEWAY)      │
                    └────────┬─────────┘
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
   DTU_001~007          DTU_008~009         DTU_010~011
   (7个电表DTU)         (2个变送器DTU)      (2个继电器DTU)
   各挂1台电表          各挂1台变送器       各挂1台继电器
```

## 逻辑拓扑（子设备关系）

```
DTU网关 (GATEWAY)
│
├── DTU_001 ─── METER_001 (总表)
├── DTU_002 ─── METER_002 (支表, parent=METER_001)
├── DTU_003 ─── METER_003 (支表, parent=METER_001)
├── DTU_004 ─── METER_004 (支表, parent=METER_002)
├── DTU_005 ─── METER_005 (支表, parent=METER_002)
├── DTU_006 ─── METER_006 (支表, parent=METER_003)
├── DTU_007 ─── METER_007 (末端, parent=METER_003)
├── DTU_008 ─── ENV_001 (变送器)
├── DTU_009 ─── ENV_002 (变送器)
├── DTU_010 ─── RELAY_001 (继电器)
└── DTU_011 ─── RELAY_002 (继电器)
```

## 数据流

```
RS485设备 → DTU节点(采集) → DTU网关(MQTT上报) → ThingsKit平台
```

## MQTT Topic

| Topic | 用途 |
|-------|------|
| v1/devices/me/attributes | 网关属性 |
| v1/gateway/telemetry | 子设备遥测（DTU代发） |

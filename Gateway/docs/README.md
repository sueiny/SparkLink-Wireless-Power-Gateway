# Gateway docs

本目录是 Gateway 项目的内部文档库，结构与 Obsidian vault `D:\Dtu\Gateway` 保持一致。

## 分区

| 分区 | 用途 |
| --- | --- |
| `00_项目说明` | 项目介绍、方案、技术栈、拓扑等对外说明 |
| `10_项目复盘` | 阶段总结、联调记录、上板测试、问题复盘 |
| `20_技术沉淀` | 线程、IPC、Socket、Modbus、数据源、物模型等技术总结 |
| `30_CodeReview` | 架构约束、review、整改记录、后续重构 |
| `40_学习技能` | 学习路线、工具技能、项目内 skill 整理 |
| `90_参考资料` | 外部资料、设备手册、ThingsKit/语雀导出 |

## 归档规则

新增文档先写入对应分区，再更新相应索引。参考资料、PDF、语雀全量导出默认只放入 `90_参考资料`，不挂到主线总览，避免淹没项目主线。

同步到 Obsidian 时，优先保持同名路径；如果内容已经在 Obsidian 中手工整理过，采用去重补齐策略，不直接覆盖。

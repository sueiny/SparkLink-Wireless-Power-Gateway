:::color4
💡 注意

平台导入大屏时，数据为json并要与格式保持一致，否则导入的大屏json错误导致界面显示为空白，所有大屏导入操作时最好先设计好大屏后进行导出功能，方便后续使用。

模板创建后，组件数据无法绑定，需要在大屏中使用该模板后重新进行数据绑定，模板只是负责界面样式。

:::

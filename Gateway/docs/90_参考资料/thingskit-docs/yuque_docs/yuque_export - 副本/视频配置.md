title: 视频配置
---

支持摄像头视频流管理，如：接入自建流媒体服务器或海康、大华等品牌的摄像头到平台进行统一管理。

# 创建视频配置
:::color4
💡 注意

视频配置分三种类型：手动输入，流媒体配置，GBT28181。对应自己需要进行添加。

视频配置如果想通过流媒体获取，则先需要在流媒体配置功能进行新增数据。[流媒体配置](https://yunteng.yuque.com/avshoi/v2xdocs/gmgulkcka3dmizzn)

GBT28181视频需要从摄像头配置连接信息到平台。[视频设备接入](https://yunteng.yuque.com/avshoi/v2xdocs/up9pq50qu8xy80u2)

视频配置需要从分屏模式切换列表模式才能编辑。

:::

点击创建视频，进行视频配置。

# 预览GBT28181
:::color4
💡 提示

当视频配置通过流媒体获取时，列表下进行预览，则可以使用云台控制进行摄像头转动，放大缩小操作。

长按按钮调用API，由于视频流媒体提供的流，有一定延时，请耐心等待。

:::

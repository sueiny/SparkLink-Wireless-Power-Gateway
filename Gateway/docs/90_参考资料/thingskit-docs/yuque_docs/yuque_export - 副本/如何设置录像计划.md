title: 如何设置录像计划
---

:::color4
💡注意：

如果设置的录像时间段，视频设备不能正常播放，录制不会成功。

由于录制过程中会有数据进行**存储**，需注意部署的服务器配置是否支持

:::

# 前置条件
视频设备和通道已正常接入平台，同时视频能正常播放。可通过[视频设备接入](https://yunteng.yuque.com/avshoi/v2xdocs/up9pq50qu8xy80u2#a3hge)进行配置

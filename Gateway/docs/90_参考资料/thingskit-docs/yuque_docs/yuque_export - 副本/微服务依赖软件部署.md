# 第1/3步：微服务部署必装软件
## 1/5：安装基础数据库
### 启动
```shell
cd /_makeFile/postgresql
sudo docker-compose	up -d                   
```

### 查看启动日志
```shell
sudo docker-compose logs  --tail=200 -f    #查看管理界面日志                                 
```

:::info
💡 注意

如果要实现数据库的高可用可以参考下列文档：

:::

[Postgresql+Pgpool II 搭建高可用数据库MD5](https://yunteng.yuque.com/to2an3/hxd0ak/gmhni98wo3zso392)

### (可选)Redis集群组网
:::color3
⚠️ 警告

REDIS集群组网时，必须在所有节点启动后，在任一REDIS环境执行如下命令对集群进行初始化。

redis-cli -a litangyuan --cluster create 节点1:6379 节点2:6379 节点3:6379

:::

```shell
sudo docker exec -it redis bash                          
```

:::info
💡 提示

需要将文本【节点n】替换后执行命令。

例如：redis-cli -a thingskit --cluster create 192.168.1.235:6379 192.168.1.236:6379 192.168.1.237:6379

:::

```shell
redis-cli -a 密码 --cluster create --cluster-replicas 1 节点1:6379 节点2:6379 节点3:6379 节点4:6379 节点5:6379
```

## 安装关系数据库Cassandra
:::info
💡 提示

环境变量【DATABASE_TS_TYPE】的值为cassandra时，<font style="color:rgb(38, 38, 38);">必须部署Cassandra。默认值：timescale。</font>

:::

### 启动
```shell
cd /_makeFile/cassandra
sudo docker-compose	up -d

                            
```

### 查看启动日志
```shell
sudo docker-compose logs  --tail=200 -f    #查看管理界面日志                                 

```

### 更新数据库用户信息
```shell
docker exec -it cassandra bash

```

```shell
cqlsh -u cassandra -p cassandra -f /docker-entrypoint-initdb.d/init_cassandra.cql                          
```

```shell
cqlsh -u cassandra ALTER ROLE cassandra WITH SUPERUSER = false AND LOGIN = false;                        
```



## <font style="color:rgb(38, 38, 38);">安装流媒体ZLMediaKit</font>
:::warning
‼️ 特别注意

该模块属于增值功能，不包含在基础功能内，部署文件需要额外获取。

:::

:::info
💡 提示

使用<font style="color:rgb(38, 38, 38);">GBT28181协议接入视频时，必须部署流媒体。流媒体部署文件获取后，直接在/_makeFile/media/下解压。</font>

:::

### 启动
```shell
cd /_makeFile/media/
docker load -i zlm.tar			#导入流媒体镜像
docker-compose	up -d   		#启动流媒体
```

### 查看启动日志
```shell
docker-compose logs  --tail=200 -f    #查看管理界面日志                                 
```

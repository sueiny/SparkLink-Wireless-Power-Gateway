# v2.0.2 Release
:::color2

## 🥇 新增功能
+ 边缘设备同步IP到云端
+ 场景联动支持设备状态联动
+ 产品、动态数据源、过滤器、第三方鉴权导入导出功能
+ 微信手机号快速登录（移动端）
+ 组态图表组件新增样式配置
+ 数据看板新增历史数据组件
+ 大屏联动组件，用于选择组织联动设备

## 🥈 功能优化
+ 定时任务调度日志表作分区处理
+ Open Api接口调用日志表作分区处理
+ 设备导入导出功能
+ 平台视频流优先采用ws_fmp4播放流
+ 组态、大屏、看板分享时关联的设备自动设置为公开
+ 大屏实时折线图组件优化
+ 移动端告警详情优化

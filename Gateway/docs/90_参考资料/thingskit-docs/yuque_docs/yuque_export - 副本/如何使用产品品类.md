---
title: 如何使用产品品类
---

      产品品类用于区分产品所属领域，并在产品引用时便于区分和使用。超级管理员账号默认有许多产品品类供租户选择使用。

:::color4
💡 提示

超级管理员创建的，可供租户管理员使用，租户管理员创建的只能自己使用。

:::

# 应用场景
产品品类常应用于拥有固定产品类型的用户，用户可以通过产品品类定义产品需要的物模型。当各租户在使用时可以直接应用标准的产品品类。这样创建的产品就自动拥有了标准产品品类的物模型。

# 产品品类的操作
## 新增产品品类
登录超级管理员点击设备管理，选择产品品类，新增品类并导入物模型数据或者自行添加物模型数据。

:::color4
💡 提示

新增产品品类之前，在字典管理中新增字典配置来用于品类中的领域选择。便于区分产品品类使用场景。

:::

## 删除产品品类
选择品类进行删除，或选择多个进行批量删除。

:::color4
💡 提示

产品品类在租户管理员中被产品选择使用后，必须先删除产品，否则不允许产品品类的删除操作。

:::

## 查看产品品类详情
点击产品品类详情，对物模型进行新增，导入和导出功能的操作。

:::color4
💡 提示

产品品类可以是json格式文件或者excel格式。

导出的物模型为json格式，导入可以是json或者excel，excel有模板下载，excel只能导入物模型属性。

:::

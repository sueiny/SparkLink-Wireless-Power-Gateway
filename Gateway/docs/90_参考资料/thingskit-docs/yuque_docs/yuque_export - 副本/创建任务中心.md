---
title: 创建任务中心
---

ThingsKit物联网平台中的任务功能非常有用，可以以间隔时间和周期时间为触发器，设定各种定时任务：

<font style="color:rgb(51, 51, 51);">通过设定定时任务，可以实现对设备的智能化管理和控制，提高设备的效率和可靠性，同时也可以降低运维成本。</font>

:::color4
💡 提示

启用任务后，绑定的设备都定时执行定时任务，可在设备详情的任务菜单中，禁用所选的设备不开启此定时执行。

任务中，目标类型可以对设备和产品进行选择，选择产品则是该产品下全部设备都会执行该任务。

:::

# 运行任务
:::color4
💡 注意

点击运行任务，手动下发一次任务命令。未启用则会提示禁用状态，不能执行。

手动执行时，选择目标类型可为全部目标设备或自行指定目标设备。

:::

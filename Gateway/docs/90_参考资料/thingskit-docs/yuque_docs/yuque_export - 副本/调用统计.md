---
title: 调用统计
---

调用统计数据呈现需要在应用配置创建后，使用测试工具Postman进行操作才能生成应用配置的接口调用历史。

首先需要准备Postman工具：

[Postman: The World’s Leading API Platform | Sign Up for Free](https://www.postman.com/)

## 可参考实际操作文档：
[ThingsKit物联网平台OpenApi文档](https://yunteng.yuque.com/avshoi/open_api)

title: 3D模型
---

<font style="color:rgb(5, 7, 59);background-color:rgb(253, 253, 254);">3D模型是在计算机图形学中，为某个表面或物体在专用软件中创建的数字形象，代表了一个物理实体在三维空间中的形态。</font>

# <font style="color:rgb(5, 7, 59);background-color:rgb(253, 253, 254);">创建3D模型</font>
点击可视化素材库，切换到3D模型，创建3D模型，填入信息后确定。

:::color4
💡 注意

3D模型只使用于可视化管理中的大屏和移动端大屏功能。

创建3D模型后，会自动跳转到3D设计界面。

:::

💡 提示

当删除应用配置时，如果此应用配置之前调用了API，并在调用记录中存在数据，删除时调用记录也一并进行了删除操作。

:::

通过调用统计文档操作生成的调用记录展示效果：[调用统计](https://yunteng.yuque.com/avshoi/v1xdocs/pixfxu4wtay7ppio)。

---
title: 组织管理
---

组织作为资源的容器，用于承载资源与划分资源的所属关系。每个设备、产品、可视化、任务等等，都需要选择所属组织，便于资源划分。

# 创建组织
:::color4
💡 提示

平台新账号都会有一个名为默认的最上级默认组织，用户可自行添加。

上级组织不能是此组织本身，组织创建后才方便新增其他平台功能的数据，如：新增设备，新增场景联动......

在有些功能里面可以直接新增组织，例如：新增设备等......

在后续数据陆续添加后，组织所关联的设备，场景联动和账号存在等，则删除组织会提示存在关联无法删除。

:::

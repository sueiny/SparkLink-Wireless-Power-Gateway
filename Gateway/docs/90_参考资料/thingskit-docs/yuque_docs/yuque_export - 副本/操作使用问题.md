# 设备状态有几种？
## 答案
总共三个状态：未激活、在线和离线。如果设备推送了数据上来，就由未激活变更为->在线，长时间没推送数据，会自动离线。设备多久离线的时间设定，可以在配置文件中修改。

Chrome、FireFox

# 左侧菜单为什么功能比较少？
:::info
💡提示

首次进入刚刚部署的平台后，会发现左侧菜单没有自己想要的功能。

:::

## 答案
请检查您登录的是不是超级管理员账号，超级管理员没有设备管理、规则引擎、数据中心、任务中心、监控运维这些功能，需要您先创建租户，然后使用租户登录平台，就可以使用这些功能了。

参考这个文档操作：

[租户列表](https://yunteng.yuque.com/avshoi/v2xdocs/yor8wfhncl6wup4l)

# 为什么报表导出没有数据？
## 答案
请检查您的报表配置的 **最近时间 **是否在设备的** 上报周期 **内，如果在周期内则说明对应时间段的设备没有上报数据，所以生成的报表导出没有数据。

# 如何修改平台logo、ico、左上角名称？
## 答案
:::info
💡提示

首次部署后 ，如果需要对平台进行个性化修改，则可以登录超级管理员账号后在平台管理内进行修改。此修改全平台租户有效，平台不支持对单个租户进行配置。

:::

```shell
账号：sysadmin
密码：Sysadmin@123
```

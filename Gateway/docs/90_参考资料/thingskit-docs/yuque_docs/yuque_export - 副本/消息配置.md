title: 消息配置
---

通过阿里云平台或者腾讯云平台，开通短信服务，在平台填入相关信息，短信配置启用，消息模版启用即可发送。

通过qq邮箱或其他电子邮箱，开通邮件发送服务，邮件配置启用，消息模版启用后即可进行发送。

# 开通及创建消息配置

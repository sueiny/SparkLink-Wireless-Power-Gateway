---
title: 源码部署包调整
---

:::color3
💡提示：

将源码打包后，需要调整部署包文件才可以启动平台。源码请找相关人员获取。文档中文件路径都使用默认路径，如果有更改，请自行调整。

:::

# 前端更改
:::color3
💡提示：

将前端文件打包后将生成的dist.zip文件解压至web_ui（管理界面）、data_view（大屏）、scada（组态）、nr（nodered）。

:::

```shell
/_makeFile/_workspace/
```

# 后端更改
:::color3
💡提示：

需要将后端部署包文件tk-4.1.0-boot-service.jar（平台运行文件）、tk-4.1.0-boot-install.jar（平台初始化文件）替换到对应的目录下并修改名称。

:::

```shell
单机
/_makeFile/_workspace/web_server/				#平台运行文件目录，文件名修改为thingsKit.jar
/_makeFile/_workspace/web_server/data/sql/		#平台初始化文件，文件名修改为initDb.jar
微服务
/_makeFile/_workspace/core_rule/				#平台运行文件目录，文件名修改为thingsKit.jar
/_makeFile/_workspace/core_rule/data/sql/		#平台初始化文件，文件名修改为initDb.jar
```

# 部署包更改
:::color3
💡提示：

需要修改启动脚本完成对源码打包文件的适配。

:::

```shell
单机
/_makeFile/thingskit/_image/java-start.sh
微服务
/_makeFile/services/_image/java-start.sh


//单机修改内容

cd $THINGSKIT_DIR/data/sql && rm -rf xjar  && go build xjar.go  &&
  ./xjar java -Dfile.encoding=UTF-8 $XJAR_OPTS -Dinstall.data_dir=$THINGSKIT_DIR/data -jar ./initDb.xjar
修改为
cd $THINGSKIT_DIR/data/sql && java -Dfile.encoding=UTF-8 $XJAR_OPTS -Dinstall.data_dir=$THINGSKIT_DIR/data -jar ./initDb.jar

rm -rf xjar  && go build xjar.go && ./xjar java -Dapp.name=$TB_SERVICE_ID $XJAR_OPTS $JAVA_OPTS -Dinstall.data_dir=$THINGSKIT_DIR/data  -Dlogging.config=$THINGSKIT_DIR/_logback.xml  -jar ./thingsKit.xjar
修改为
java -Dapp.name=$TB_SERVICE_ID $XJAR_OPTS $JAVA_OPTS -Dinstall.data_dir=$THINGSKIT_DIR/data  -Dlogging.config=$THINGSKIT_DIR/_logback.xml  -jar ./thingsKit.jar

//微服务修改内容

cd $DATA_DIR/sql && rm -rf xjar  && go build xjar.go  && ./xjar java -Dfile.encoding=UTF-8 $XJAR_OPTS -Dinstall.data_dir=$DATA_DIR -jar ./initDb.xjar
修改为
cd $DATA_DIR/sql && java -Dfile.encoding=UTF-8 $XJAR_OPTS -Dinstall.data_dir=$DATA_DIR -jar ./initDb.jar

rm -rf xjar  && go build xjar.go && ./xjar java -Dapp.name=$TB_SERVICE_ID $JAVA_OPTS $XJAR_OPTS -Dinstall.data_dir=${DATA_DIR} -Dlogging.config=/thingskit/logback.xml  -jar ./thingsKit.xjar
修改为
java -Dapp.name=$TB_SERVICE_ID $JAVA_OPTS $XJAR_OPTS -Dinstall.data_dir=${DATA_DIR} -Dlogging.config=/thingskit/logback.xml  -jar ./thingsKit.jar
```

# 创建定时任务
点击新增，输入定时任务相关信息，确认。

# 类型
定时任务类型：默认，报表，任务中心。

:::color4
💡 提示

超级管理员新增的定时任务为默认。

租户管理员：任务中心生成定时任务为任务中心类型[创建任务中心](https://yunteng.yuque.com/avshoi/v2xdocs/ig2lsbbhzea1rd0p) ，报表配置生成定时任务为报表类型[如何配置](https://yunteng.yuque.com/avshoi/v2xdocs/scni7bzt85i0p3eb)。

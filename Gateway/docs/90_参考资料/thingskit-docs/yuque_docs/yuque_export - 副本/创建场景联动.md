:::color4
💡 提示

触发类型：简单（不需要多余触发条件）、持续时长（当第一次触发，如果持续时长时间内都达到触发器值，则触发器生效）、重复次数（当第一次触发到设置的重复次数都超过条件筛选值，则触发器生效）

在场景联动产生告警发送消息之前，需要对消息模板、消息配置、告警联系人、告警配置进行新增，并且全部设置为启用状态。这样在场景联动触发告警时，及时发送短信，邮件和钉钉通知到用户，并进行处理。

:::

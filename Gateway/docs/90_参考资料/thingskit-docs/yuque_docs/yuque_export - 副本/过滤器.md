---
title: 过滤器
---

<font style="color:rgb(5, 7, 59);background-color:rgb(253, 253, 254);">用于数据过滤或处理的脚本程序。这种脚本可以运行在服务器端或客户端，根据特定的规则或条件来筛选、修改或转换数据。通过编写和运行过滤器脚本，可以高效地处理和分析数据，为业务决策提供有力支持。</font>

# <font style="color:rgb(5, 7, 59);background-color:rgb(253, 253, 254);">创建过滤器</font>
:::color4
💡 注意

过滤器可在动态数据源和第三方鉴权配置中使用，源数据通过不同的过滤器可以实现不同的结果。

当动态数据源和第三方鉴权配置中使用过滤器时，删除该过滤器时会有相应的提示，并不能进行删除，需要解除使用该过滤器后方可删除操作。

:::

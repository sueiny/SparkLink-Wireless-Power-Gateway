title: 如何进行国标级联
---

:::color4
💡注意：

部署时thingsboard.yml里面的GBT28181_SIP_IP环境变量配置要与实际IP一致，否则国标级联连不上。

:::

# 前置条件
国标级联需要对SIP认证有一定了解，如遇问题可咨询售前或技术进行支持。

# 上级平台
:::color4
💡注意：

创建设备的访问令牌要与下级平台的SIP编码一致，否则上级平台关联的设备不能上线。

:::

# Modbus（Modbus-RTU、Modbus-TCP）
:::color4
💡注意：

在添加物模型时，由于Modbus协议的特殊性。建议物模型连续地址长度有 **差异 **，否则解析会有误差。

例如：<font style="color:rgb(51, 51, 51);">保持寄存器</font>40001-40006 为一个连续地址，产品里面就不能再定义有长度与其一致的连续物模型地址。

**缩放因子**，填写为**10**

如进行数据上报，数据为十六进制的值 **00A5 **➡️ **165 **/** 10 **得出结果** 16.5**

如进行命令下发，数据为十进制值的值 **15.2 **➡️ **15.2 * 10** 得出结果 **0098**

:::

# 前提条件
创建[消息配置](https://yunteng.yuque.com/avshoi/v2xdocs/vcy6xqi6z2z5corg)。

# 创建消息模板
点击新增消息模板按钮，选择短信或邮件并填入相关参数，确认新增。

:::color4
💡 提示

模板用途短信适用全部，邮件适用于告警，钉钉适用于告警，语音适用于告警，企业微信适用于告警。

同一模板用途不可重复启用。

:::

# 使用示例
超级管理员中，消息配置和消息模板（登录用途）创建好并启用，此时用手机发送短信验证码的方式登录租户管理员。

<img src="https://cdn.nlark.com/yuque/0/2025/png/36222522/1756716534848-d5fad8dd-8128-41e6-a6e4-f7024f2a1615.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VGhpbmdzS2l0%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10" width="1911" title="" crop="0,0,1,1" id="ue69cd6a7" class="ne-image">

# <font style="background-color:rgb(253, 253, 254);">概述</font>
<font style="background-color:rgb(253, 253, 254);">编辑器的作用是让用户可以通过简单的配置和拖拽，创建出符合自己需求的网页界面。</font><font style="color:rgb(44, 62, 80);">您可以快速搭建已接入设备的数据可视化展示，以及可交互的控制界面。</font>

## <font style="background-color:rgb(253, 253, 254);">编辑器的特点</font>
+ <font style="color:rgb(44, 62, 80);">简单易用的可视化布局。</font>
+ <font style="color:rgb(44, 62, 80);">数据绑定轻松搞定，无需任何代码。</font>
+ <font style="color:rgb(44, 62, 80);">支持几十个常用的看板组件，样式比较丰富。</font>
+ <font style="color:rgb(44, 62, 80);">响应式布局。</font>
+ <font style="color:rgb(44, 62, 80);">支持共享模式，可生成开放 URL。</font>
+ <font style="color:rgb(44, 62, 80);">不断推出新看板组件。</font>

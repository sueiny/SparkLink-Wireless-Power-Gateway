# 视频管理

## 概述

ThingsKit视频管理模块支持监控和管理摄像头设备，接入海康、大华等品牌摄像头，支持GB/T28181国标协议。

## 支持的接入方式

### GB/T28181 国标协议
- 标准协议接入
- 支持海康、大华、宇视等主流品牌
- 支持NVR/DVR接入

### RTSP/RTMP
- 标准流媒体协议
- 第三方摄像头接入
- 自定义流地址

### ONVIF
- 设备发现
- PTZ控制
- 配置管理

## 功能特性

### 实时预览
- 多画面分屏（1/4/9/16分屏）
- 高清/标清切换
- 全屏显示
- 画面截图

### 录像回放
- 按时间检索录像
- 录像下载
- 倍速播放（0.5x/1x/2x/4x）
- 时间轴拖拽

### 云台控制（PTZ）
- 方向控制（上下左右）
- 变焦控制（放大缩小）
- 光圈控制
- 预置点调用
- 巡航路线

### 告警联动
- 移动侦测告警
- 视频丢失告警
- 设备离线告警
- 联动抓图/录像
- 联动通知

## 摄像头接入配置

### GB28181接入配置
```json
{
  "protocol": "gb28181",
  "deviceId": "camera_001",
  "name": "大厅主摄像头",
  "sipServer": {
    "ip": "192.168.1.100",
    "port": 5060
  },
  "sipDevice": {
    "id": "34020000001320000001",
    "domain": "3402000000",
    "username": "admin",
    "password": "****"
  },
  "video": {
    "resolution": "1920x1080",
    "framerate": 25,
    "codec": "h264"
  }
}
```

### RTSP接入配置
```json
{
  "protocol": "rtsp",
  "deviceId": "camera_002",
  "name": "门口摄像头",
  "url": "rtsp://admin:****@192.168.1.101:554/Streaming/Channels/101",
  "transport": "tcp"
}
```

## 视频流处理

### 转码
- H.264/H.265转码
- 分辨率调整
- 码率控制

### 转协议
- RTSP → HLS
- RTSP → FLV
- GB28181 → WebRTC

### 存储
- 本地存储
- 云存储
- 循环录像
- 按事件录像

## 视频与设备联动

### 告警联动配置
```yaml
name: 移动侦测告警
trigger:
  type: video
  event: motion_detection
  cameraId: camera_001
action:
  - type: snapshot
    quality: high
  - type: record
    duration: 30s
  - type: notify
    channel: [wechat]
    template: "检测到移动活动"
  - type: device_control
    deviceId: alarm_light_001
    command: flash
```

### 视频叠加
- 设备数据叠加显示
- 告警信息叠加
- 时间戳叠加
- 自定义文字叠加

## Web播放器

### 集成方式
```html
<iframe src="/video/player?cameraId=camera_001" width="640" height="480"></iframe>
```

### API接口
```
GET /api/v1/video/stream/{cameraId}
POST /api/v1/video/control/{cameraId}
GET /api/v1/video/record/{cameraId}
```

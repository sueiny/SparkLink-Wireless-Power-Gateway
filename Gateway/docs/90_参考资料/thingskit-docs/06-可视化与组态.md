# 可视化与组态

## 大屏设计器

### 概述
拖拽式可视化大屏设计器，无需编码即可完成数据大屏。支持图表、地图、视频、控件等100+组件。

### 组件库

#### 图表组件
| 组件 | 说明 |
|------|------|
| 折线图 | 趋势数据展示 |
| 柱状图 | 分类数据对比 |
| 饼图 | 占比数据展示 |
| 散点图 | 分布数据展示 |
| 雷达图 | 多维数据对比 |
| 仪表盘 | 单值数据展示 |
| 进度条 | 进度数据展示 |
| 水球图 | 比例数据展示 |

### 数据绑定

#### 静态数据
```json
{
  "type": "static",
  "data": [10, 20, 30, 40, 50]
}
```

#### 设备数据
```json
{
  "type": "device",
  "deviceId": "sensor_001",
  "field": "temperature",
  "timeRange": "1h"
}
```

#### API数据
```json
{
  "type": "api",
  "url": "/api/v1/statistics",
  "method": "GET",
  "refreshInterval": 5000
}
```

### 布局方式

#### 自适应布局
- 基于百分比定位
- 适配不同分辨率
- 响应式设计

#### 栅格布局
- 基于栅格系统
- 组件自由拖拽
- 精确像素定位

### 大屏配置
```json
{
  "name": "智慧园区大屏",
  "width": 1920,
  "height": 1080,
  "backgroundColor": "#0a1628",
  "backgroundImage": "/images/bg.jpg",
  "refreshInterval": 5000,
  "components": [
    {
      "type": "line-chart",
      "x": 100,
      "y": 100,
      "width": 600,
      "height": 400,
      "dataSource": {...}
    }
  ]
}
```

## Web组态设计器

### 概述
低代码可视化Web组态设计器，采用标准HTML5技术，支持工业级组态组件库。

### 工业组件库

#### 基础组件
| 组件 | 说明 |
|------|------|
| 管道 | 流体管道展示 |
| 阀门 | 各类阀门（球阀、蝶阀、闸阀） |
| 泵 | 水泵、油泵等 |
| 储罐 | 立式、卧式储罐 |
| 仪表 | 压力表、温度表、流量计 |
| 电机 | 各类电机 |
| 变频器 | 变频器控制 |

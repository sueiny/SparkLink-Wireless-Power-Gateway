## 二、测试指标

### 2.1 核心指标

| 指标 | 定义 | 目标 | 计算方式 |
|------|------|------|----------|
| **丢包率** | 发送失败比例 | 8%~10% | (sent - success) / sent |
| **平均延时** | 成功 SEND 的 ACK 延迟 | 200ms~300ms | avg(source_to_target_ms) |
| **路由可达性** | 所有节点是否可达 | 100% | 无 route_unreachable |
| **P95 延时** | 95 分位延迟 | - | sorted(lats)[95%] |

### 2.2 丢包率计算说明

```
丢包率 = (实际SEND - 成功) / 实际SEND

其中：
- 计划轮次 = 节点对数 × rounds（默认 90对 × 2 = 180）
- 路由不可达 = 路由找不到路径（不计入 SEND）
- 实际SEND = 计划轮次 - 路由不可达
- ACK timeout = SEND 后未收到 ACK
- 成功 = 实际SEND - ACK timeout
```

**关键**：路由不可达不计入丢包率分母！所以丢包率 = ACK timeout / 实际SEND。

### 2.3 延迟组成

```
total_latency_ms = inference_ms + source_to_target_ms

其中：
- inference_ms = Dijkstra 算法计算时间（~0.2ms）
- source_to_target_ms = SEND 到 ACK 的往返时间（200-3000ms）
```

**注意**：在 single_send 模式下，`source_to_target_ms` 就是 `latency_ms`（SEND→ACK 时间）。

### 2.4 每轮数据字段（rounds.jsonl）

```json
{
  "target": 5,
  "source": 4,
  "route": [0, 4, 5],
  "success": true,
  "latency_ms": 201.5,

  "total_latency_ms": 201.7,
  "inference_ms": 0.16,
  "status": "success",
  "round_index": 1
}
```

## 三、RSSI 拓扑分析

### 3.1 查看拓扑

```bash
python3 -c "
import json
with open('logs/dijkstra_hw/第N次测试/原始JSON数据/state.json') as f:
    state = json.load(f)
print(f'节点: {sorted(state[\"nodes\"])}')
print(f'边数: {len(state[\"edges\"])}')
# 检查节点入度
from collections import Counter
in_deg = Counter(e['dst'] for e in state['edges'])
for n in sorted(in_deg):
    print(f'  节点{n:02X}: 入度={in_deg[n]}')
"
```

### 3.2 检查双向边

```bash
python3 -c "
import json
with open('logs/dijkstra_hw/第N次测试/原始JSON数据/state.json') as f:
    state = json.load(f)
edges = set((e['src'], e['dst']) for e in state['edges'])
bidi = sum(1 for (a,b) in edges if (b,a) in edges)
print(f'总边: {len(edges)}, 双向: {bidi}, 单向: {len(edges)-bidi}')
"
```

## 四、延迟分析

### 4.1 按首跳分析

```bash
python3 -c "
import json
from collections import defaultdict
with open('logs/dijkstra_hw/第N次测试/原始JSON数据/rounds.jsonl') as f:
    lines = f.readlines()
by_hop = defaultdict(list)
for l in lines:
    d = json.loads(l)
    if d.get('success') and d.get('route') and (d.get('source_to_target_ms') or 0) > 0:
        hop = d['route'][1] if len(d['route']) > 1 else d['route'][0]
        by_hop[hop].append(d['source_to_target_ms'])
for h in sorted(by_hop):
    lats = by_hop[h]
    print(f'via {h:02X}: n={len(lats)} avg={sum(lats)/len(lats):.0f}ms')
"
```

### 4.2 按跳数分析

```bash
python3 -c "
import json
from collections import defaultdict
with open('logs/dijkstra_hw/第N次测试/原始JSON数据/rounds.jsonl') as f:
    lines = f.readlines()
by_hops = defaultdict(list)
for l in lines:
    d = json.loads(l)
    if d.get('success') and d.get('route') and (d.get('source_to_target_ms') or 0) > 0:
        hops = len(d['route']) - 1
        by_hops[hops].append(d['source_to_target_ms'])
for h in sorted(by_hops):
    lats = by_hops[h]
    print(f'{h} hops: n={len(lats)} avg={sum(lats)/len(lats):.0f}ms median={sorted(lats)[len(lats)//2]:.0f}ms')
"
```

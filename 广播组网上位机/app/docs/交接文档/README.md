# 交接文档目录

本目录包含 Mesh 自组网上位机项目的完整交接文档，供接手的 agent 参考。

## 文档清单

| 文件 | 内容 |
|------|------|
| `01_项目背景.md` | 硬件环境、协议格式、目录结构、RSSI 拓扑采集原理 |
| `02_项目操作方式.md` | 环境准备、Dijkstra/D3QN 测试命令、日志结构、代码修改指南 |
| `03_项目进度与目标.md` | 目标指标、已完成工作、最优参数、未完成工作 |
| `04_测试方式和指标.md` | 测试流程、丢包率计算、延迟组成、RSSI 分析方法 |
| `05_关键注意点.md` | Dongle 操作、拓扑波动、路由算法、常见陷阱 |
| `06_代码结构详解.md` | 核心文件说明、数据结构、参数传递链 |
| `07_最新测试结果.md` | 最优参数、测试结果、权重调整记录、代码修改记录 |

## 快速开始

1. 阅读 `01_项目背景.md` 了解项目全貌
2. 阅读 `02_项目操作方式.md` 学习操作方法
3. 阅读 `05_关键注意点.md` 避免踩坑
4. 使用 `03_项目进度与目标.md` 中的最优参数开始测试

## 当前最优参数

```bash
python -m pc_dijkstra_cli bench \
  --port /dev/ttyUSB0 --dongle-addr 16 --rounds 2 \
  --send-mode single_send --rssi-seconds 20 --rssi-requests 10 \
  --max-hops 3 --no-retry --ack-timeout 5.0 --hop-penalty 8 --interval 2.0
```
